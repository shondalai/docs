# Reports and Analytics

The Reports screen aggregates store performance across sales, inventory, customers, taxes, and shipping. It is reached from the admin sidebar at **Reports**.

## Date Range

A single date-range selector applies to every tab:

- Today
- Yesterday
- Last 7 Days
- Last 30 Days (default)
- This Month
- Last Month
- This Year
- Custom Range

Each tab automatically compares the selected window against the equivalent prior period.

## Tabs

| Tab | Contents |
|-----|----------|
| Overview | Headline KPIs and quick links to other tabs |
| Sales | Gross sales, net sales, refunds, discounts, taxes, shipping |
| Orders | Order counts by status, average order value |
| Products | Top sellers, low performers, units sold, revenue by product |
| Customers | New, returning, and total customers; lifetime value distribution |
| Subscriptions | Active, new, cancelled, MRR, ARR, churn |
| Coupons | Usage count, redemptions, discount given |
| Tax | Tax collected by rate and jurisdiction |
| Shipping | Shipping revenue, breakdown by method |
| Reviews | Volume, rating distribution, verified ratio |
| Export | One-click exports of any report as CSV or XLSX |

Plugins can contribute additional tabs between Shipping and Reviews.

## Sales Report

| Metric | Description |
|--------|-------------|
| Gross Sales | Revenue before refunds and tax |
| Net Sales | Revenue after refunds |
| Refunds | Total refunded in the window |
| Discounts | Coupon discounts applied |
| Taxes | Tax collected |
| Shipping | Shipping revenue |

The report includes a revenue trend chart with the comparison period overlaid.

## Orders Report

Order counts by status and average order value, with a chart for each status over time.

## Products Report

Top and bottom performers by units sold and revenue. Also includes category-level breakdown.

## Customers Report

New customers, returning customers, total customers, and a chart of new sign-ups over time. The Customer Value section ranks customers by spend.

## Subscriptions Report

Active, new, and cancelled counts, plus MRR and ARR. The growth chart plots subscription count and MRR over time.

## Tax and Shipping Reports

Tax collected by rate and shipping revenue by method, useful for accounting reconciliation.

## Export

The Export tab and the Export button in each report header download the active filter view. Supported formats are CSV and XLSX.
