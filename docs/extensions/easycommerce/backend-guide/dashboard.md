# Dashboard

The dashboard opens by default when you enter EasyCommerce. It shows period-over-period activity, operational alerts, top products, and recent orders. Long-range analytics live in [Reports](reports.md).

## Period Snapshot Cards

Three cards summarize trailing activity against the matching prior window:

| Card | Compared Against |
|------|------------------|
| **Today** | Yesterday to date |
| **This Week** | Last week to date |
| **This Month** | Last month to date |

Each card shows revenue, order count, and a percent delta. A delta of `new` means there was no comparable activity in the prior window.

## Alerts

When pending orders or low-stock products exist, an alert strip surfaces them with direct links to the filtered Orders or Products list.

## Sales Chart

A revenue area chart spans the trailing 30 days. Hovering shows the daily revenue and order count for that point.

## Top Products

The five best-selling products for the current period, by units sold and revenue.

| Column | Description |
|--------|-------------|
| Product | Name and thumbnail |
| Units | Quantity sold |
| Revenue | Total revenue |

## Recent Orders

The eight most recent orders.

| Column | Description |
|--------|-------------|
| Order | Order number (links to detail) |
| Customer | Customer name and email |
| Status | Status badge |
| Total | Order total in the order's currency |
| Date | Order placed date |
