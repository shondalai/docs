Sociable ships with several modules and plugins, the following is an explanation about them.

## System Plugin

Sociable System plugin is used to redirect the users to Sociable login and registration pages. Enable this plugin if you want to use Sociable built-in login and registration forms.

When to enable this plugin?

- If you want to override your default Joomla user login and registration pages and
- If you want to use Sociable login and registration forms

## Authentication Plugin

Sociable has built-in support for logging in via third-party social platforms such as Facebook, Twitter, Google, Linkedin, etc. Enable this plugin to use social authentication plugins. Make sure to enable System plugins to enable Sociable login/registration pages.

When to enable this plugin?

- If you want to provide social login functionality that comes bundled with Sociable
- If you want to override default Joomla login and registration forms.

## Content Plugin

Sociable has built-in support for Joomla articles to support awarding points, streaming activity about reading and posting articles, assigning badges, etc. Make sure to enable this plugin to allow Sociable to integrate with article related actions.

When to enable this plugin?

- If you want to integrate your articles with Sociable points system, for example, award points when the user read an article
- If you want to integrate your articles with Sociable activity streams, for example, show activity about a new article on the Sociable activity page.
- If you want to integrate your articles with Sociable badge system, for example, award a badge to the user when he post 10 articles, or read 100 articles etc.

## User Plugin

User plugin is one of the important plugins which shall be enabled in order to allow Sociable function correctly. The user plugin synchronizes new users to Sociable, supports referral system, integrates with Sociable points and activity stream systems.

When to enable this plugin?

- Always enable this plugin

## Sociable Apps

### Core Profile Apps

- **Activity:** Shows the user activity tab on the user profile page
- **Badges:** Shows badges earned by the user on the user profile pages
- **Connections:** Shows user connections tab
- **Fields:** Shows user profile custom fields
- **Photos:** Shows user photos and albums
- **Points:** Shows points history
- **Referrals:** Shows referrals list

### Third-party Profile Apps

- **Articles:** Shows list of articles posted by the user
- **Kunena:** Shows user topics posted on Kunena component
- **Polls:** Shows list of polls submitted by the user with Community Polls component
- **Questions:** Shows list of questions submitted by the user with Community Answers component
- **Quizzes:** Shows the list of quizzes submitted by the user with Community Quiz component
- **Quotes:** Shows list of quotes submitted by the user with Community Quotes component
- **Topics:** Shows list of topics submitted by the user with CjForum component
- **Tracks:** Shows list of tracks submitted by the user with GPS Tools component
