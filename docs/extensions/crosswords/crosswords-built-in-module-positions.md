You can use below module positions to insert modules in crosswords pages.

- cw_list_above_navigation &#8211; On top of the listings and categories
- cw_list_below_navigation &#8211; below categories but above listings
- cw_details_above_title- Above crossword title on details page
- cw_details_below_navigation &#8211; Just above the crossword table
