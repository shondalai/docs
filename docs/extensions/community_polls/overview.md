# Community Polls
Create and manage polls on your Joomla website.
## Overview
Community Polls is a powerful Joomla extension that allows you to create engaging polls and gather user opinions.
## Key Features
- Poll Builder - Create polls with multiple options
- Real-time Results - Display live poll results
- User Voting - Secure voting system
- Results Analytics - Track voting patterns
## Getting Started
Start by adding your documentation files to the docs/extensions/community_polls folder.
