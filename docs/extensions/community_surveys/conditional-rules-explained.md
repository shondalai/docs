## Introduction

Community Surveys has a powerful feature to alter the flow of the survey based on many factors using conditional rules. Conditional rules can be created at each question.

Conditional rules can be setup on choice and grid type questions.

## Actions to Trigger Conditional Rules

The following actions can trigger a conditional rule

- User select a specific answer of a question
- User do not select a specific answer of a question
- User select any answer
- User select none of the answers

## Outcome from the Conditional Rules

The following outcomes are possible when a conditional rule is triggered

- End survey
- Show certain questions on same page
- Skip the flow to a different page

## Creating Conditional Rules

To create a conditional rule, follow the below steps

- Go to Components -> Community Surveys -> Surveys
- Click on edit link of your survey to edit questions
- Add a question by clicking on any question type on toolbar
- Click on Conditional Rules tab
Here you will see 4 options
- **If user answer this question** &#8211; if you want to take action when user selected ANY answer of this question, select this option
- **If user do not answer this question** &#8211; if you want to take action when user selected NONE of the answers of this question, select this option
- **If user select an answer/column** &#8211; if you want to take action when user selected SPECIFIC answer which you choose, select this option
- **If user do not select an answer/column** &#8211; if you want to take action when user DO NOT select SPECIFIC answer which you choose, select this option

Based on the option you select from above 4, the following options will be shown
- **Skip to page:** &#8211; Selecting this option will take the user to selected page of the survey. For example, skip to page 6 of survey.
- **Show question on this page** &#8211; This option will show chosen question on same page when the criteria met
- **Finalize response** &#8211; This will finishes the survey
