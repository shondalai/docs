# Rewardify
Gamification and rewards system for your Joomla website.
## Overview
Rewardify adds gamification to your Joomla site with points, badges, and rewards.
## Key Features
- Points System
- Badges
- Rewards
- Leaderboards
## Getting Started
Add your documentation to docs/extensions/rewardify folder.
