# Community Answers
Q&A platform for your Joomla website.
## Overview
Community Answers is a Stack Overflow-style Q&A extension for Joomla.
## Key Features
- Questions & Answers
- Voting System
- Reputation System
- Best Answer Selection
## Getting Started
Add your documentation to docs/extensions/community_answers folder.
