# Community Quotes
Share and display quotes on your Joomla website.
## Overview
Community Quotes allows users to submit, rate, and share inspirational quotes.
## Key Features
- Quote Submission
- Categories
- Ratings
- Quote Display
- Comments (powered by QuillThreads)

## Comments
Quote discussions are powered by [QuillThreads](/quillthreads/overview). To enable comments under your quotes, install the QuillThreads package and enable the **QuillThreads - Community Quotes** plugin (it ships with Community Quotes). Quotes then show a full threaded discussion with moderation, spam protection, and optional AI checks. If QuillThreads is not installed, quotes simply display without a comment thread.

## Getting Started
Add your documentation to docs/extensions/community_quotes folder.
