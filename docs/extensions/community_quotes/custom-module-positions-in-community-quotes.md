Community Quotes support built in module positions to which you can assign your modules like Adsense ads or any custom module.

Below are the module position supported.

- **quotes-common-above-navbar** &#8211; On all pages, above the toolbar
- **quotes-common-below-navbar** &#8211; On all pages, below the toolbar
- **quotes-list-above-categories** &#8211; On list pages, above the categories listing block
- **quotes-list-below-categories** &#8211; On list pages, below the categories listing block
- **quotes-list-above-pagination** &#8211; On list pages, above the pagination
- **quotes-list-below-pagination** &#8211; On list pages, below the pagination
- **quote-details-below-quote** &#8211; On quote details page, below the quote details
- **quote-details-above-comments** &#8211; On quote details page, above comments section
