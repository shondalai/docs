All corejoomla extensions use CjLib language files for translating date strings.

To translate the date strings shown on front-end, translate the file administrator/language/en-GB/en-GB.com_cjlib.sys.ini to your language.

You can also find community-contributed language translations on the Transifex site.
[https://www.corejoomla.com/support/language-translations.html](https://www.corejoomla.com/support/language-translations.html)
