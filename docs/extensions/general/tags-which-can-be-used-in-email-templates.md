The documentation describes the smart tags that can be used in email templates. The tags will be replaced automatically with the necessary content before the emails are sent to users.

These placeholders can be added at any place in the email templates of the respective extension.

## CjForum

**In subject:**

- `{TOPIC_TITLE}` &#8211; this tag will be replated with actual topic title

**In description:**

- `{NAME}` &#8211; Receiver name
- `{AUTHOR_NAME}` &#8211; Name of the user who performed an activity such as creating topic, replying, like or dislike etc.
- `{TOPIC_TITLE}` &#8211; Title of the topic
- `{TOPIC_DESCRIPTION}` &#8211; Description of the topic
- `{TOPIC_URL}` &#8211; Url of the topic
- `{CATEGORY}` &#8211; Name of the topic category
- `{SITENAME}` &#8211; Your site name as defined on global configuration
