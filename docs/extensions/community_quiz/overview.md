# Community Quiz
Create and manage quizzes on your Joomla website.
## Overview
Community Quiz is a comprehensive Joomla extension for creating interactive quizzes, tests, and assessments.
## Key Features
- Quiz Builder - Create quizzes with multiple question types
- Automated Scoring - Automatic grading and feedback
- Certificates - Award certificates to quiz takers
- Analytics - Track quiz performance and results
## Getting Started
Add your documentation files to the docs/extensions/community_quiz folder.
