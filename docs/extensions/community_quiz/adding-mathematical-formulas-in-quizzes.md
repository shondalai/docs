Starting v5.0.3, you can add mathematical formulas inside the quiz description, question description, and answers. Community Quiz uses KaTex library to display mathematical formulas and it ships with a copy of it.

### How to enable formulas?

- Make sure you have v5.0.3 or later version installed
- Enabled &#8220;Load Maths/Tex Library&#8221; option either in the component options or in the quiz options
- If you would like to insert the formulae using your Joomla editor, enable &#8220;Button &#8211; Maths&#8221; (type editors-xtd) plugin is enabled.

### How to add formulas

- If you enabled &#8220;Button &#8211; Maths&#8221; plugin, a button is displayed in your Joomla editor to easily add formulas.
- You can also add formulas manually anywhere in the description or the answers using the HTML code shown below

<span class="math">YOUR LaTex FORMULA HERE</span>
