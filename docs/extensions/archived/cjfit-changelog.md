**v1.0.3 (2018-01-14)**

+ New option to show selected article on dashboard page when profile is not connected
^ Disable email settings by default in activity plugin
^ Allow deleting challenges
^ Updated default character set of tables to utf8mb4_unicode_ci
* Activity stream description of personal goals contains placeholder text instead of actual values
* Username in activity stream is linked to dashboard page instead of user profile
* Activity stream shows actual value as goal value and vice versa

**v1.0.2 (2017-12-17)**

+ Show user achievements listing on dashboard page
+ Added support for weekly and monthly leaders in leaderboard module
* Renamed the leaderboard module folder name to avoid conflict name with other modules (reconfiguration required)
* CSS adjustments of leaderboard module

**v1.0.1 (2017-12-13)**

* Fitbit updates are not updated by the cronjob
* User dashboard urls showing 404 error when there is no dashboard menu item present
