Community-driven fitness and wellness app for Joomla!. Fitbit integration with challenges, profiles, personal goals and more.

Features

- Create challenges to encourage users to do more workouts
- Personal goals
- Show user workouts in a dashboard
- Reward users for reaching personal goals and winning challenges
- Social integrations
- Email notifications
