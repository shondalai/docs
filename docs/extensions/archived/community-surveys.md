Built on latest Web technologies, Community Surveys is the perfect tool for your business in conducting surveys. Powerful features, skip logic, great looking UI makes it a perfect survey tool for your website.

## System Requirements

CoreJoomla software runs fine on any webserver with minimum system resources. All it require the following software installed.

- Joomla 3.x, Joomla 3.6 or above is recommended. Joomla 1.5/2.5 versions are no longer maintained/supported.
- PHP 5.5 or above
- MySQL 5
- CjLib Component
