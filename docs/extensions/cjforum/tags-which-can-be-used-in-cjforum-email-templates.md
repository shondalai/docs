CjForum sends emails using templates that you define in the backend. The following tags can be used in your email templates.

**In subject:**

- `{TOPIC_TITLE}` &#8211; this tag will be replated with actual topic title

**In description:**

- `{NAME}` &#8211; Receiver name
- `{AUTHOR_NAME}` &#8211; Name of the user who performed an activity such as creating topic, replying, like or dislike etc.
- `{TOPIC_TITLE}` &#8211; Title of the topic
- `{SITENAME}` &#8211; Your site name as defined on global configuration
