# CjForum
Discussion forum for your Joomla website.
## Overview
CjForum is a full-featured forum extension for Joomla.
## Key Features
- Forum Categories
- Topics & Posts
- Moderation Tools
- User Permissions
## Getting Started
Add your documentation to docs/extensions/cjforum folder.
