CjForum topics can be shown in Joomla! articles using the content plugin bundled with the CjForum package.

## Enabling Content Plugin

- Go to Extensions -> Plugins
- Filter the plugins with type as Content
- Enable &#8220;Content &#8211; CjForum&#8221; plugin

## Using Content Plugin

Now you can embed the desired topic into your Joomla! articles. Add the below shortcode anywhere in your article content.

`{DISCUSSTOPIC }`

The topic will be automatically displayed when you view the article.
