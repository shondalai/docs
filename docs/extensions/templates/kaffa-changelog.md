# Kaffa Template Changelog

All notable changes to Kaffa Template will be documented in this file.


## [1.0.0] - 2026-07-14

### 🚀 Added
- Initialized Kaffa Template extension
- Added modern split-panel 404/403/500 and offline pages
- Included footer payment method chips
- Adopted rich footer skin from in-tree work
- Skinned the Studio mega menu
- Implemented token-driven skeleton themed to template
- Set hero CTA to fall back to shop route when featured section
- Used shop route instead of dead #ts-featured anchor when section is off or empty
- Clamped to 32 to prevent breaking square cards and added new
- Displayed featured product as rotated typographic bag label, pure-type batch
- Added names, italic category lines, dashed feet and round add buttons

### 🐛 Fixed
- Addressed review findings from adversarial pass
- Header no longer traps fixed elements as containing block
