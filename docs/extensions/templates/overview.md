# Shondalai Templates Documentation

Shondalai Templates is a collection of seven commerce-ready Joomla templates, all built on the shared Template Studio framework. This documentation covers choosing a template, installing it, configuring it visually, building custom pages, and managing your licence.

## The collection at a glance

Every template in the collection is a complete storefront design for Joomla. All seven run on the same Template Studio engine, so once you have learned to configure one you can configure any of them. Your custom pages and your configuration workflow carry over when you switch templates.

Each template installs as a single Joomla package containing:

- The site template itself
- The shared Template Studio framework
- The Studio Pages page builder component
- The Shondalai License Manager plugin
- A system plugin that powers the live preview
- A sample data plugin that builds a full demo store in a few clicks

## What Template Studio gives you

Template Studio is a visual configuration app that opens when you edit your template style in **System → Site Template Styles**. Instead of long plain forms, you work in a tabbed app with a full-screen live preview of your site.

| Area | What you can do |
|------|-----------------|
| Design presets | Pick a ready-made colour and typography preset, then override the accent colour, font pairing, corner radius, and light or dark colour scheme |
| Brand | Set your store name, tagline, logo, and a separate logo for dark surfaces |
| Layout | Choose a header style, sticky header, top bar with announcement text and social icons, boxed or full-width layout, and product grid density |
| Navigation | Build mega menu panels with links, categories, products, banners, HTML, or module columns, and attach icons to menu items |
| Modules | Arrange your template's module positions on a 12-column layout grid |
| Storefront home | Let the template take over your home page with hero, trust bar, featured products, category tiles, story, and newsletter sections |
| Footer | Build footer link columns, legal links, a brand blurb, and payment method badges |
| Studio Pages | Build custom pages such as About, FAQ, and campaign landings from ready-made sections, themed automatically by your active template |
| Advanced | Add custom CSS, head and body scripts, an analytics ID, and extra body classes |
| License | Activate your Shondalai licence to unlock automatic updates through the Joomla updater |

## Requirements

| Requirement | Minimum |
|-------------|---------|
| Joomla | 5.4 or newer, including Joomla 6 |
| PHP | 8.1 or newer |
| EasyCommerce | Optional, see the note below |

:::note EasyCommerce is optional
Every template works as a standard Joomla template without EasyCommerce. You need EasyCommerce installed and enabled for the store features: the storefront home page, cart and account actions in the header, shop category columns in mega menus, the demo store sample data, and the Product grid section in Studio Pages.
:::
