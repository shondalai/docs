# Template Gallery

Seven site templates, one shared framework. This page introduces each template's design personality, its demo store, and the colour presets it ships with, so you can pick the right starting point for your store.

All seven templates are built on the same Template Studio framework. That means they share the same configuration model (brand, navigation, layout, footer), the same storefront integration, and the same Studio Pages page builder. Switching from one template to another does not change how you work: the tools stay identical, only the design language changes. Each template brings its own typography, surfaces and colour presets, and each ships with sample data that builds a complete demo store in a few clicks.

## Forge

Forge is a precision workshop template with an editorial, monochrome heart. Its signature look is luxury restraint: a serif display face, square corners, no shadows, and ink on white. It suits brands that sell craftsmanship and want the product photography to do the talking.

The demo store is **Forge Works**, a workshop-grade hand tool shop with the tagline "Tools built for the second generation".

| Preset | Accent | Feel |
|--------|--------|------|
| Editorial | Ink black (#0a0a0a) | Luxury monochrome. Zero radius, no shadow, editorial restraint. |
| Studio | Indigo (#6366f1) | Clean SaaS neutral. Rounded and calm, great for digital goods and subscriptions. |

:::note Two very different moods
Forge's monochrome personality lives in the Editorial preset. The Forge Works demo store runs the Studio preset instead, so the demo looks rounded and indigo rather than black and white. Try both before you decide.
:::

## Pulse

Pulse is an energetic direct-to-consumer template built around a bold coral accent. Rounded cards, soft shadows and a confident grotesque display face give it a conversion-forward, street-ready feel. Its second preset flips the mood entirely to a warm, serif boutique.

The demo store is **Pulse Supply**, a street-ready activewear and basics brand.

| Preset | Accent | Feel |
|--------|--------|------|
| Vibrant (default) | Coral (#ff4d2e) | Commercial and energetic. Rounded, soft shadows, conversion-forward. |
| Atelier | Warm copper (#b5663a) | Warm boutique. High-contrast serif on warm paper, pill buttons. |

The demo store is **Kaffa Roasters**, a specialty coffee roastery.

## Switching templates

You can install several templates side by side and change your mind at any time. Set the one you want as the default style under **System → Site Template Styles** and your site switches instantly: Studio Pages, the storefront and your content all re-theme automatically to the new design.

:::tip Each style keeps its own configuration
Template Studio configuration is stored per template style, not shared across templates. When you switch, the new template starts from its own brand, menu and footer settings, so set those up in the new style before making it the default. Your pages and store content carry over untouched.
:::
